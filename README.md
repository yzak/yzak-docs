# Yzak Apps Documentation

This repository contains documentation, support pages, and privacy policies for all Yzak apps.

## Apps

### Simple Counter App
- [Support Page](https://yzak.github.io/yzak-docs/simple-counter-app/)
- [Privacy Policy](https://yzak.github.io/yzak-docs/simple-counter-app/privacy.html)

## Structure

```
yzak-docs/
├── simple-counter-app/
│   ├── index.html (support page)
│   └── privacy.html (privacy policy)
└── [future-app]/
    ├── index.html
    └── privacy.html
```

## Adding New App Documentation

1. Create a new directory with your app name
2. Add `index.html` for support page
3. Add `privacy.html` for privacy policy
4. Update this README with links
